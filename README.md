# AquaTrack

Cross-platform hydration tracking app built with Kotlin Multiplatform.

## Overview
AquaTrack helps users track their daily water intake, set personal hydration goals, and receive smart reminders.  
The app is designed with clean architecture principles for scalability and maintainability.

## Features
- Add daily water intake
- Set personal hydration goals
- Visualize daily progress
- Receive smart notifications

## Architecture
- MVVM (Model-View-ViewModel) for clean separation of concerns
- Shared business logic in Kotlin Multiplatform
- Local storage using Room / SQLDelight

## Tech Stack
- Kotlin Multiplatform
- Compose Multiplatform
- Coroutines & Flow
- Local database
